# Typing Speed Test

A terminal-based typing speed test application that measures your typing speed in words per minute (WPM) and accuracy. It allows you to select difficulty levels and keeps track of your personal records.

## Features
- Select difficulty level: Easy, Medium, Hard
- Measure WPM and accuracy
- Store personal records

## Installation
1. Clone this repository.
2. Install Python if you haven't already.

## Usage
1. Run the application using the command:
   ```bash
   python typing_test.py
   ```
2. Follow the on-screen instructions to select a difficulty and start typing.

## Records
Your personal records will be stored in `records.json`.
